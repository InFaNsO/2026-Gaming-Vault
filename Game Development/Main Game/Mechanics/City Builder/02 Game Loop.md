# City Builder — Game Loop

> Two interlocking loops: a **zoom-in** cozy town-planning loop and a **zoom-out** production/logistics-network loop. Mani threads through both.

---

## Zoom-in loop — plan a cozy town (Town-to-City soul)

`place functional buildings (grid) → meet household needs via Mani utilities + Mani-tech goods → raise desirability (decoration, amenities, landscape, adjacency) → households grow / new families arrive → plan ahead (leave room) → tier up (Settlers → Citizens → Artisans)`

- Needs (consumed, hard) + Wants (decorative, soft) drive happiness.
- Services = needs/wants met by proximity (radius). Decoration = free-build on top of the grid.
- Money: passive tax from happy households + trade income − building upkeep (+ light tax↔happiness lever).

## Zoom-out loop — run the network (Anno brain)

`harvest raw Mani → refine (building) → split: utilities + Mani-tech goods → higher-tier goods need multiple elements → found/supply towns in other biomes → ship via logistics (roads → boats → Vayu-freight → Akash portals) → balance the portfolio so no town starves`

- Each biome-map = 1 big home town + 1–2 mining outposts (DSP model).
- Multi-element goods **force inter-biome trade** → the logistics web is the challenge.
- Building-expansion-via-Mani raises output toward ceilings.

## Progression loop — climb the eras

`research (labs → Knowledge) → unlock buildings / decorations / expansions / recipes → master an element → era advances → next biome unlocks → repeat → all 4 mastered → Akash Synthesis → endgame`

- One unified staircase: population tier + biome unlock + logistics tier + goods complexity advance together.
- Tech tree (single Knowledge currency, element-branched) culminates in Akash.

---

> The **zoom-in** loop is the cozy heart; the **zoom-out** loop is the strategic engine; the **progression** loop is the arc. See [[00 City Builder]] for the full doc map.
